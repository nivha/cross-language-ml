iDempiere Business Suite  es un software ERP Sistema de planificación de recursos empresariales de código abierto, que incluye tambien funcionalidades de CRM  Administración de la Relación con los Clientes y SCM Administración de la cadena de suministro.

==Funcionalidad==
Adempiere cuenta con todas las funciones que un ERP moderno debe tener, permite ser configurado para cada caso especifico de negocio, lo que obviamente requiere un plan de implementación muy cuidadoso.

iDempiere soporta:

*Múltiples Organizaciones
*Múltiples Grupos Empresariales
*Múltiples Idiomas
*Múltiples  Monedas
*Múltiples Estructuras de Contabilidad

==Arquitectura==
iDempiere se derivó del proyecto ADempiere el cual a su vez se derivó de la versión libre de Compiere. Por tanto hereda entre otras características el soporte para base de datos Oracle y PostgreSQL, JAVA como lenguaje de programación, pero principalmente el  diccionario de aplicación del proyecto Compiere™. que extiende el concepto de "Diccionario de Datos" hacia la administración en iDempiere de entidades, reglas de validación, ventanas, formatos, lógica de despliegue, etc. Haciendo que muchas funciones puedan realizarse directamente en el diccionario, sin necesidad de modificar el código JAVA.

== Tecnología==
La primera gran diferencia de iDempiere con su versión precedente (ADempiere), esta en la actualización tecnológica de su plataforma, de esta manera esta basado en un framework OSGi permitiendo una estructura de plug-ins, con el servidor de aplicacion Apache Tomcat como remplazo de jboss mejorando el rendimiento y tamaño, Eclipse Buckminster y una migración de ZK3.6 a ZK6 sobre el cual se implementó una interface rediseñada.
